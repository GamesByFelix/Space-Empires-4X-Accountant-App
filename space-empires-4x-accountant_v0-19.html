Keyboard Shortcuts:
E Scroll to Economics table.
Shift-N New Phase.
F Focus first text field of current phase.
Arrows  Text field value +/-1 or +/-5.
M. Wheel  Text field value +/-1. Hold Shift for +/-5.
Esc Unfocus text field or close windows.
Tab Focus next text field.
T Scroll to Tech table.
S Scroll to Ship Tech table.
Shift-P Toggle "Auto-Purchase".
Shift-S Toggle "Show All Groups".
D Roll dice.
N Show notes.
H Toggle Help/Settings window.
Ctrl-E  Export game data.
Ctrl-I  Import game data.
Ctrl-P  Reset game data (purge).

Tips, Tricks & More:
- Only enter numbers in the text fields, not plus/minus signs.
- You can drag the Info and Dice windows around at the top.
- Learn the keyboard shortcuts above.
- Backup your game data with the export button above.

- Use the browser zoom to change the text size (Ctrl+/-).
- Set your browser to fullscreen mode for a better experience (F11).
- Save this HTML file for offline use (Ctrl-S).

- This web app does not transmit any data.
- All data is stored in the local browser cache.
